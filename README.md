### Installation

* Install [FFmbeg](https://ffmpeg.org/)
``` Terminal
sudo apt install ffmpeg
```

* Run config.py 
```terminal
python3 config.py
```

* In config.ini add path to your music dir
```ini
[PATH] 
OUTPUT = <path>   EX: /home/development/Music

```


### Usage 

> Type command  "download" in the terminal
